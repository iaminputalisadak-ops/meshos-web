Your PHP server is not running. The easiest solution is to install XAMPP.

Step 1: Download XAMPP

1. Click the button below to download XAMPP

Download XAMPP

2. Choose: XAMPP for Windows (PHP 8.x version)

Step 2: Install XAMPP

1. Run the installer

2. Select components: Apache, MySQL, PHP

3. Install to default location: C:\xampp

4. Complete the installation

Step 3: Start Apache

1. Open XAMPP Control Panel

2. Click START button next to Apache

3. Wait until it shows "Running" (green)

Step 4: Move Your Files

Option A: Copy backend folder

From: C:\Users\km450\Downloads\startup\backend

To: C:\xampp\htdocs\backend

Option B: Move entire project

From: C:\Users\km450\Downloads\startup

To: C:\xampp\htdocs\startup

Step 5: Access Admin Panel

Open browser and go to:

http://localhost/backend/admin/index.php

Login:

Username: admin

Password: admin123

  • Keep XAMPP Control Panel open while using admin panel

Alternative: Use XAMPP's PHP for Built-in Server

If you want to use port 8000 instead:

1. Open Command Prompt

2. Navigate to backend folder

3. Run:

C:\xampp\php\php.exe -S localhost:8000

4. Access: http://localhost:8000/admin/index.php
